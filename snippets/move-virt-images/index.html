How to move libvirt images into /home directory

August 29, 2016

Objectives

Do not waste space in / partition, because it’s usually too small.

Step by step solution

  • Create new directory /home/dir and move images
    • # mv /var/lib/libvirt/images/* /home/dir/
  • Set SElinux context for files in this directory
    • Check the context of libvirt directory: semanage fcontext -l | grep virt_image
    • Use the same for the new directory: semanage fcontext -a -t virt_image_t "/home/virtstorage(/.*)?"
    • restorecon -R /home/virtstorage/
  • Add alternative storage pool
    • virsh pool-dumpxml default > pool.xml
    • Change name, delete uuid (but leave tags there) and change the path
    • virsh pool-create pool.xml
    • virsh pool-refresh name

Example:

<pool type='dir'>
    <name>newname</name>
    <uuid></uuid>
    <capacity unit='bytes'>1234</capacity>
    <allocation unit='bytes'>1234</allocation>
    <available unit='bytes'>1234</available>
    <source>
    </source>
    <target>
        <path>/home/dir</path>
    </target>
</pool>
