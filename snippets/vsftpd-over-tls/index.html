FTP over TLS with vsftpd

August 29, 2016

How to set up vsftpd over SSL/TLS

  1. Install vsftpd:

    # <pkg manager> <install command> vsftpd openssl

  2. Generate self signed certificate 1

    sudo openssl req -x509 -nodes -days 365 -newkey rsa:1024 -keyout /etc/ssl/private/vsftpd.pem -out /etc/ssl/private/vsftpd.pem

  3. Configure vsftpd.conf:

    ssl_enable=YES
ssl_sslv2=NO
ssl_sslv3=NO
ssl_tlsv1=NO
ssl_tlsv1_1=NO
ssl_tlsv1_2=YES
# ^^^ Only available on CentOS
rsa_cert_file=/etc/ssl/private/vsftpd.pem
rsa_private_key_file=/etc/ssl/private/vsftpd.pem
allow_anon_ssl=NO
force_local_data_ssl=YES
force_local_logins_ssl=YES

  4. Run vsftpd service

How to test it

Since SSL is used, netcat cannot be used anymore, but openssl provides something similar:

$ openssl s_client -connect localhost:21 -starttls ftp -tls1_2
